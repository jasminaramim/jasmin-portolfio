# Jasmin Ara Mim - Premium Portfolio

![Portfolio Banner](https://jasmin-v01.vercel.app/favicon.ico)

This is the premium portfolio website for Jasmin Ara Mim, a Full Stack Developer. It features a modern, interactive, and fully dynamic 3D user interface powered by React and Framer Motion. 

## 🌐 Live Website
**[https://jasmin-v01.vercel.app](https://jasmin-v01.vercel.app)**

## 🚀 Features
- **Fully Dynamic Content:** Manage hero section, projects, experience, education, skills, services, and statistics via a secure Admin Dashboard.
- **Dynamic 3D & Floating UI:** Features floating orbit tech icons and interactive developer code cards.
- **Modern Stack:** Built with React, TypeScript, Tailwind CSS, Framer Motion, and Vite.
- **Backend:** Express API with MongoDB for data management.
- **Responsive Design:** Optimized for all screen sizes and mobile devices.

## 🛠️ Tech Stack
- Frontend: React.js, Tailwind CSS, Framer Motion, Lucide Icons
- Backend: Node.js, Express.js
- Database: MongoDB
- Deployment: Vercel

## 🔒 Admin Dashboard
The portfolio includes an integrated admin panel to manage all data dynamically without changing the code.

**Login URL:** `/admin/login`
